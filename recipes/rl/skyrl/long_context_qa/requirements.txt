# requires). This gives you the pip-installed `skyrl` package; the launcher

# IMPORTANT: install WITH the `skyrl-train` extra. The launcher entrypoint

# the training deps that extra pulls (deepspeed / hydra-core / jaxtyping /
# torchdata / accelerate / wandb / …). Installing bare `skyrl` resolves an env
# that fails at import time (`ModuleNotFoundError: jaxtyping`, then torchdata).

# FlashAttention-2 built against torch 2.10 (== vLLM 0.18.0's torch pin).
# The `skyrl-train` extra requires `flash-attn==2.8.3`; SkyRL's pyproject
# `[tool.uv.sources]` redirects that to this exact torch-2.10 wheel — but
# `[tool.uv.sources]` is ONLY honored by `uv run`/`uv sync` from the SkyRL
# repo, NOT by `uv pip install skyrl[...] @ git+...`. Without this explicit pin,
# the resolver picks the default torch-2.11 wheel, whose ABI is incompatible
# with torch 2.10 (ImportError: undefined symbol _ZNK3c104cuda10CUDAStream5queryEv)
# and breaks both the trainer's FA2 path and vLLM's rotary-embedding import.
flash-attn @ https://github.com/lesj0610/flash-attention/releases/download/v2.8.3-cu12-torch2.10-cp312/flash_attn-2.8.3%2Bcu12torch2.10cxx11abiTRUE-cp312-cp312-linux_x86_64.whl ; python_version == "3.12" and platform_machine == "x86_64"
